Troubleshooting
---------------

You can run unit tests I have prepared like this (install pytest_ first):


.. code:: sh

  # use your package manager to install the package "pytest"
  # here, I examplify with "pip":
  $ pip install pytest
  $ pytest rr
    ...

    rr/test_analysis.py ....                                                [100%]

   =============================== 4 passed in 0.18s ================================
